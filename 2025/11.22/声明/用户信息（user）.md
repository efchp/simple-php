# 语法
```
get(内容)
```
# 内容
## 链接类
以https://www.efchp.com/example/add.php?id=100&value=niubi举例

| 内容       | 结果                                                       | 解释        |
| -------- | -------------------------------------------------------- | --------- |
| protocol | https                                                    | 协议        |
| domain   | www.efchp.com                                            | 域名        |
| path     | /example/add.php                                         | 路径        |
| uri      | /example/add.php?id=100&value=niubi                      | 获取包含参数的路径 |
| link     | https://www.efchp.com/example/add.php?id=100&value=niubi | 完整链接      |
## 其他

| 内容  | 解释   |
| --- | ---- |
