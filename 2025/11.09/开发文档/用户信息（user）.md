# cookie
## 声明
所有仅针对键和值，域名为访问页面的域名，时间为1天，路径为根目录
## 单个
### 增
```
cookie("add",键,值);
```
### 删
```
cookie("delect",键,值);
```
### 改
```
cookie("set",键,新值);
```
### 查
```
cookie("get",键);
```
## 批量
### 增
```
cookie("add",数组);
```
### 删
```
cookie("delete");
```
### 改
将对应的键所对应的值进行修改（不涉及的键不处理）
```
cookie("set",数组);
```
### 查
以数组的形式获取
```
cookie("get");
```
